# CSS box-sizing attribute with example given blew:
1. box-sizing: border-box;
2. box-sizing: content-box;



## 1. box-sizing: border-box;
```
.box {
    width: 500px;
    height: 500px;
    border: 1px solid red;
    box-sizing: border-box;
    padding: 50px;
    margin: 20px;
    text-align: justify;
}
```

## 2. box-sizing: content-box;
