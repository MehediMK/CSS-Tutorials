# All flex align-items attribute given below with examples: 


 1. align-items: baseline;

## example:
 ```
 .container {
    width: auto;
    height: auto;
    background-color: gainsboro;
    font-size: 25px;
    display: flex;
    flex-direction: row;
    justify-content: center;
    align-items: baseline;
}
```
 
 2. align-items: center;
 
## example:
 ```

    font-size: 25px;
    display: flex;
    flex-direction: row;
    justify-content: space-evenly;
    align-items: center;
}
```
